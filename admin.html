Iframe Google Maps personnalisé

Collez directement une URL d'embed depuis Google Maps

Non configuré
Comment obtenir l'URL d'embed ?
Sur Google Maps → cliquez sur votre lieu → Partager → Intégrer une carte → copiez uniquement l'URL du champ src="…".

Aperçu en direct :

Si une URL est enregistrée ici, elle sera utilisée à la place de la clé API Google Maps.

Supprimer l'URL reviendra à utiliser la clé API (ou OpenStreetMap).
