1. Présentation Générale

Le site Miralocks est une plateforme web moderne conçue pour la gestion complète d'un institut capillaire spécialisé. Elle combine une vitrine marketing esthétique avec un panneau d'administration puissant et sécurisé.

Vanilla HTML5/CSS3/JS Supabase (PostgreSQL) Google Analytics 4 Looker Studio Edge Functions

2. Architecture Système

A. Frontend (Côté Client)

Le site est construit sans framework lourd (React/Vue) pour garantir une rapidité absolue et un référencement naturel (SEO) optimal. Le fichier main.js pilote les interactions légères, tandis que supabase.js centralise toutes les communications avec la base de données.

B. Backend (Côté Serveur)

La puissance du site repose sur Supabase, qui gère :

  • Base de données : Stockage des rendez-vous, articles, photos et logs.
  • Authentification : Accès sécurisé à l'administration par email/mot de passe.
  • Storage : Hébergement des photos envoyées par les clients lors des réservations.
  • Edge Functions : Envoi automatique des emails de confirmation et rappels WhatsApp.

3. Modules Principaux

Système de Rendez-vous

Le client choisit une prestation, une date et peut joindre une photo de ses cheveux. Une fois soumis, le rendez-vous apparaît dans le planning admin. Un email de confirmation est envoyé automatiquement et une notification arrive sur l'admin.

Panneau d'Administration

Un tableau de bord complet permettant de :

  • Gérer les articles de Blog, la Galerie Photos et les Vidéos.
  • Modérer les avis clients.
  • Visualiser le trafic réel via l'intégration Looker Studio.
  • Consulter l'historique complet des actions et des connexions.

Sécurité Renforcée

Chaque connexion admin est tracée (Adresse IP + Type d'appareil). Les tentatives de "forcer" le mot de passe sont enregistrées et signalées en rouge dans l'historique pour prévenir toute intrusion.

4. Fonctionnalités Spéciales

A. Traduction Multilingue

Le site utilise Google Translate via js/lang.js. Un script personnalisé "masque" les logos et éléments publicitaires de Google pour préserver l'élégance du site tout en offrant la traduction instantanée.

B. Statistiques & Analytics

Plus de statistiques "maison" lourdes. Le site est relié à Google Analytics 4. Le rapport visuel est intégré dans l'admin via une iframe Google Looker Studio pour une lecture simple et claire.

5. Maintenance & Évolutivité

Le site est conçu pour être "stateless". Toutes les ressources sont sur Supabase. Pour mettre à jour une prestation ou un prix, l'administrateur utilise simplement l'onglet "Prestations" dans son panneau de contrôle.
